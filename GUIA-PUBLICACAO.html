ServiçoJá

🚀 Guia de Publicação do PWA

Siga os passos abaixo para colocar o ServiçoJá no ar de graça em menos de 30 minutos.

1

Criar conta no GitHub ✅ Grátis

O GitHub é onde você vai guardar todos os arquivos do app de graça.

🔗 github.com — Clique em "Sign up" e crie sua conta
  1. Acesse github.com e clique em Sign up
  2. Crie sua conta com seu email
  3. Clique em New repository
  4. Nome: servicoja
  5. Marque como Public
  6. Clique em Create repository
2

Fazer upload dos arquivos

Envie todos os arquivos HTML do ServiçoJá para o repositório.

  1. No seu repositório, clique em uploading an existing file
  2. Arraste todos os arquivos .html, manifest.json, sw.js e icon.svg de uma vez
  3. Role a página, escreva "Primeiro upload" no campo de mensagem
  4. Clique em Commit changes
⚠️ Importante: certifique-se que o arquivo index.html está incluído — ele é a porta de entrada do app.
3

Publicar no Netlify ✅ Grátis

O Netlify pega seus arquivos do GitHub e coloca no ar com link público.

🔗 netlify.com — Clique em "Sign up with GitHub"
  1. Acesse netlify.com e faça login com sua conta GitHub
  2. Clique em Add new site → Import an existing project
  3. Escolha GitHub e selecione o repositório servicoja
  4. Deixe tudo como está e clique em Deploy site
  5. Aguarde 1 minuto — seu app vai ganhar um link como:
https://servicoja-seu-nome.netlify.app

Você pode personalizar o nome do link em Site settings → Change site name para algo como servicoja-sp.netlify.app

4

Configurar recebimento de cadastros por email ✅ Grátis

Use o Formspree para receber os cadastros de clientes e profissionais diretamente no seu email.

🔗 formspree.io — Crie uma conta gratuita
  1. Acesse formspree.io e crie sua conta
  2. Clique em New Form
  3. Crie dois formulários: "Cadastro Cliente" e "Cadastro Profissional"
  4. Copie o código de cada formulário — será algo como:
https://formspree.io/f/SEU-CODIGO-AQUI
  1. Nos arquivos cadastro-cliente.html e cadastro-profissional.html, adicione esse link no botão de finalizar cadastro
💡 Plano gratuito: 50 envios por mês. Suficiente para validar os primeiros usuários!
5

Configurar Google Planilhas para gerenciar pedidos ✅ Grátis

Enquanto o app não tem backend real, você gerencia os pedidos numa planilha simples.

  1. Acesse sheets.google.com e crie uma nova planilha
  2. Crie 3 abas: Clientes, Profissionais, Pedidos
  3. Cada cadastro que chegar por email você adiciona na planilha
  4. Quando um profissional demonstrar interesse, você conecta os dois pelo WhatsApp
📱 Dica: Crie um número de WhatsApp Business exclusivo do ServiçoJá para fazer as conexões de forma profissional.
6

Divulgar e captar os primeiros usuários

Com o app no ar, comece a divulgar para validar o negócio:

  • Poste o link nos grupos de WhatsApp de bairros de São Paulo
  • Crie uma página no Instagram para o ServiçoJá
  • Ofereça gratuidade total nos primeiros meses para atrair profissionais
  • Peça para amigos e familiares testarem e darem feedback
  • Poste em grupos de Facebook de bairros da sua cidade
🎯 Meta inicial: 20 profissionais cadastrados e 10 pedidos realizados. Com isso você prova que o modelo funciona!
7

Instalar o app no celular

O PWA permite que qualquer pessoa instale o app no celular sem precisar da App Store:

  • Android (Chrome): Acesse o link, aparece um banner "Instalar ServiçoJá" — toque em Instalar
  • iPhone (Safari): Toque no botão de compartilhar ⬆️ → "Adicionar à Tela de Início"

O ícone do ServiçoJá aparece na tela inicial como um app normal! 📱

🎉

Tudo pronto para o lançamento!

Com esses 7 passos você tem o ServiçoJá no ar, funcionando, instalável no celular e recebendo cadastros — sem gastar nada.
